# 🚀 Bilingual Goal Tracker (Python)

Este es mi primer proyecto como programador autodidacta. Es una herramienta de consola diseñada para gestionar mis objetivos de aprendizaje de **programación** e **inglés** simultáneamente.

---

## 🎯 Project Purpose / Propósito del Proyecto
I created this tool to track my progress and practice my English technical vocabulary.
*Cree esta herramienta para seguir mi progreso y practicar mi vocabulario técnico en inglés.*

## 🛠️ Features / Características
- **Persistence:** Goals are saved in a `.txt` file. (Los objetivos se guardan en un archivo .txt).
- **Bilingual Interface:** English menu to practice daily. (Interfaz bilingüe para practicar diario).
- **CRUD Operations:** Add, view, and delete goals. (Agregar, ver y borrar objetivos).
